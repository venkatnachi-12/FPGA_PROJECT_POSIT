The project implementation can be done in VIVADO by the following steps:
   Uploading the design source files namely, posit_multiplier.v and fp_benchmark_top.v
   Uploading the testbench files namely,posit_multiplier_tb.v under simulation sources
   Uploading the appropriate constraints file(.xdc) file
   Depending on whichever we want whether the floating point multiplier to be synthesised or the posit multiplier to be synthesised we set either of the files as top using the 'Set as Top' instruction by right clicking on the file in vivado.
   Then Running synthesis and implementation will help us to run the project
